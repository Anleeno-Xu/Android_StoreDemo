# 书店
### A book store demo

#### 实现了用户登录注册（第一次登录成功后自动登录）和信息修改、图书详情查看及搜索、图书分类查看、商品加入购物车、订单生成与删除、收藏图书、基于图灵机器人的小客服等基本模块。另外实现了图书轮播展示（含点击事件）、RecyclerView瀑布流图片展示、Fragment+ViewPager的导航框架等功能。
